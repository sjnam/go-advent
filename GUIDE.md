# 콜로설 케이브 어드벤처 — 공략 가이드 (한글판)

처음 하는 친구도 동굴을 헤매다 보물을 찾는 재미가 이 게임의 핵심이에요.
**먼저 아무 정보 없이 직접 헤매 보길** 권합니다. 막히거나 친구들과 같이
끝까지 깨고 싶을 때 이 가이드를 펼쳐 보세요.

> 명령은 한두 단어 한글입니다. 마법 주문(`xyzzy` `plugh` `plover`
> `fee`/`fie`/`foe`/`foo`)만 영어로 칩니다.

---

## 1. 가장 먼저 알아둘 것

- **램프가 생명줄**: 동굴 안은 어두워서 램프(`켜`)가 꺼지면 구덩이에 빠져
  죽기 쉽습니다. 램프엔 수명이 있으니 "램프가 어두워지고 있어" 경고가 뜨면
  슬슬 보물을 챙겨 나올 준비를 하세요. (자판기에서 배터리를 살 수 있어요.)
- **저장은 없습니다**: 원작 그대로라 세이브가 없어요. 죽으면 최대 세 번까지
  부활(`예`)할 수 있지만, 횟수가 점수에 영향을 줘요.
- **두 단어, 순서 자유**: `가져가 열쇠` = `열쇠 가져가`.
- **막히면**: `봐`(주변 다시 보기), `소지품`(가진 것), `도움말`, `점수`.
- **보물의 목적**: 보물을 **건물(집)에 가져다 놓아야** 만점입니다. 보기만
  해도 약간의 점수는 줍니다.

---

## 2. 출발 — 보급품 챙기기

길 끝 건물에서 시작합니다.

```text
안                  (건물 안으로)
가져가 열쇠
가져가 램프
가져가 음식
가져가 병
```

이제 동굴로 들어가는 길은 두 가지예요.

- **마법 지름길**: 건물에서 `xyzzy` 를 외치면 동굴 속 잔해 방으로 순간이동.
- **정공법**: `밖`으로 나가 시냇물을 따라 내려가다(`하류`) 격자문을 만나면
  `아래`로 내려가 `열어 창살`(열쇠 필요) 후 `아래`.

동굴에 들어가면 바로 `켜`서 램프를 켜세요.

---

## 3. 새와 새장 — 뱀을 쫓는 열쇠

깊은 동굴의 **산속 왕의 홀**에는 초록 뱀이 길을 막고 있어요. 뱀은 못 죽이고
못 쫓습니다 — **새**가 필요해요.

1. `cobbles`(자갈 통로) 근처에서 **새장**을 `가져가 새장`.
2. **검은 막대**(잔해 방)는 새를 겁주니, 새가 있는 방에선 **막대를 내려놓고**
   새를 잡으세요.

   ```text
   버려 막대            (새 방에서)
   가져가 새            (새장이 있어야 잡힘)
   가져가 막대
   ```

3. 뱀 앞에서 `버려 새` → 새가 뱀을 쫓아냅니다.

---

## 4. 마법 주문 정리

| 주문 | 효과 |
| --- | --- |
| `xyzzy` | 건물 ↔ 잔해 방 순간이동 |
| `plugh` | 건물 ↔ Y2 방 순간이동 (보물 나르기에 최고) |
| `plover` | 알코브 ↔ 플로버 방 (단, **에메랄드 외엔 아무것도 못 들고** 통과) |
| `fee` `fie` `foe` `foo` | 거인의 방에서 차례로 외치면 황금 알이 나타남 |

`plugh`로 Y2와 건물을 오가며 보물을 건물에 차곡차곡 모으는 게 기본 동선이에요.

## 5. 보물 15선과 획득법

| 보물 | 위치 / 해법 |
| --- | --- |
| **금괴**(금) | 너깃 방. 단, 계단 위로는 못 가져가니 다른 길로 운반 |
| **다이아몬드** | 틈의 서쪽(wfiss) |
| **은괴** | 남북 통로(ns) |
| **보석** | 산속 왕의 홀 남쪽 방 |
| **동전** | 산속 왕의 홀 서쪽 방 |
| **황금 알** | 거인의 방에서 `fee fie foe foo` |
| **삼지창** | 폭포가 있는 동굴 |
| **명나라 꽃병** | 동양식 방. **그냥 내려놓으면 깨져요** → 6장 참고 |
| **에메랄드** | 플로버 방. `plover`로만 안전하게 반출 |
| **백금 피라미드** | 어둠의 방. (플로버 방/어둠의 방에선 줍기 쉬워 해적 표적은 아님) |
| **진주** | 거대한 조개를 **삼지창으로** 열면 굴러 나옴(아래 방에서 줍기) |
| **페르시아 양탄자** | 용이 깔고 있음 → 7장 참고 |
| **향신료** | 화산 근처 바위 방(chamber) |
| **황금 사슬** | 텅 빈 방의 곰을 묶은 사슬 → 7장 참고 |
| **보물 상자** | 해적이 훔쳐 미로에 숨김 → 8장 참고 |

## 6. 핵심 퍼즐

**수정 다리 (틈 건너기)**
틈(efiss/wfiss)은 너무 넓어 못 뛰어넘어요. 틈 가에서 **막대를 흔들면**
수정 다리가 생깁니다.

```text
흔들어 막대
```

**식물 키우기 (두 구덩이 방)**
서쪽 구덩이 바닥의 작은 식물에 **병의 물을 부으면** 콩나무가 자라요.
물이 없으면 위로 가서 채우고(`채워 병`) 돌아오세요.

```text
부어 물             (식물이 있는 구덩이에서)
```

자란 콩나무를 타고 `위`로 올라가면 새 지역이 열립니다.

**꽃병 깨먹지 않기**
명나라 꽃병은 바닥에 그냥 놓으면 깨져요. **벨벳 베개**(폭신한 방)를
먼저 가져와 베개가 있는 곳에 내려놓으면 안전합니다.

**조개 → 진주**
거대한 조개(대합)는 **삼지창**을 든 채 `열어 대합` 하면 진주가 굴러
나옵니다. (들고 열면 안 되니 내려놓고 여세요.)

---

## 7. 위험한 동물들

**용 (양탄자)**
용은 양탄자를 깔고 앉아 있어요. 무기로는 안 통하고 — 맨손으로 덤비면 됩니다.

```text
공격 용
예                  ("맨손으로?"에 "예"라고 답하면 용이 죽고 양탄자가 풀림)
```

**트롤 다리 (통행료)**
트롤은 보물을 하나 던져 줘야 다리를 건네줍니다. **가치 없는 걸 던져
미끼로** 쓰고 건넌 뒤, 곰으로 쫓아내면 보물을 돌려받을 수 있어요.

```text
던져 알             (예: 황금 알을 미끼로 — 나중에 fee fie foe foo로 재소환)
```

**곰 (사슬)**
텅 빈 방의 곰은 사슬에 묶여 있어요. **음식을 먹이면** 온순해집니다.

```text
먹여 곰             (음식을 든 채)
열어 사슬           (사슬이 풀림 → 황금 사슬 보물)
가져가 곰           (곰을 데리고 다닐 수 있음)
```

순해진 곰을 트롤 다리로 데려가면 트롤이 겁먹고 달아나, 던졌던 보물을
되찾을 수 있습니다.

**난쟁이와 해적**

- **난쟁이**: 칼을 던져요. **도끼**를 주워 `던져 도끼`로 맞서세요.
- **해적**: 들고 있던 보물을 훔쳐 미로 속 **보물 상자**에 숨깁니다. 화내지
  마세요 — 덕분에 상자(보물!)의 위치가 생기고, 나중에 찾아오면 됩니다.

---

## 8. 미로 헤쳐나가기

두 종류의 미로가 있어요.

- **"전부 똑같이 생긴" 미로**: 진짜 미로. 물건을 하나씩 **떨어뜨려 표시**하며
  지도를 그리세요. (`버려 음식` 등 — 나중에 회수)
- **"전부 다르게 생긴" 미로**: 설명이 미묘하게 달라요. 해적의 보물 상자가
  이쪽에 있습니다. 설명 차이를 잘 읽어 길을 외우세요.

---

## 9. 엔드게임 — 동굴 폐쇄

**보물 15개를 모두 찾으면**(건물에 다 못 옮겼어도) 잠시 뒤 "곧 동굴이
닫힙니다"라는 음산한 목소리가 들리고 폐쇄가 시작됩니다. 이때부터는 밖으로
나갈 수 없고, 마지막 퍼즐의 방으로 옮겨져요.

마지막 방에는 검은 막대(다이너마이트)가 있습니다. 적절한 곳에서
`폭파`하면 비밀이 풀리고 — 운이 좋으면 **그랜드마스터** 등급까지 노릴 수
있어요. (이 마지막 수수께끼는 직접 풀어보는 재미로 남겨둘게요!)

---

## 10. 점수 팁

- 보물은 **건물에 가져다 놓아야** 만점(보기만 하면 일부 점수).
- 죽지 않고, 그만두지 않을수록 점수가 높아요.
- **힌트는 점수를 깎습니다** — 정 막힐 때만.
- 동굴 깊이 들어가고, 폐쇄 단계까지 도달하는 것 자체가 큰 점수예요.
- 진행 중 언제든 `점수`로 현재 점수를 확인할 수 있어요.

만점은 350점입니다. 친구들과 누가 더 높은 등급에 오르는지 겨뤄 보세요!
